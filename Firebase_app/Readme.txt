This is simple a drawing app,in which you can draw with the help of different colors,since this app is in firebase the updation of data happened in realtime.For eg run the given drawing_app file in chrome as well as in Mozilla,now draw something  in one browser and you will see the effect in the other browser automatically,this is the power of firebase,this is good it gives us the real time working scenario.The database is updated in real time. Many users can work on same tool at the same time.
