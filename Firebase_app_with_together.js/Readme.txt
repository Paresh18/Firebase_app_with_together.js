This is the same firebase app with adding some line of together.js code which give us real time collaboration on working with tool, the main advantage of using together.js is we can know the name of person with which we are working ,we can chat with him while working,we can see the cursor of the person on the tool.

Procedure 

Step 1->Add these line of code in the above html code and run it on some server.

Step2 ->As soon as you run the html file ,you will click the together.js button and a link will be generatedpass this link to your team member and ask him to open it the new tab and allow him to join the session.That’s it ,now you can see that multiple users can draw the same file at the same time.

Step 3->If you want to see the database you can click on the link and can see the updated database in json format.
